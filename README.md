# 承興 vs iEvents 架構對照頁

互動式網頁，比對承興展覽電子票券系統報價單需求 vs Iware 既有 iEvents 切版，給設計師與程式團隊參考。

## 內容

- 10 大報價單項目（$359,999.85 含稅）
- 約 80 個子功能，逐項標注
- 四色狀態：可用 / 部分可用 / 需新做 / 隱藏
- 每張卡片附「備註給程式」

## 本地開啟

```bash
open index.html
```

## 部署

靜態單頁網站，Vercel 根目錄即可。
